# Luxury Real Estate Website Design Guidelines

## Design Approach
**Reference-Based Approach** - Drawing inspiration from premium real estate leaders like Sotheby's International Realty and high-end architectural firms. This luxury real estate website requires sophisticated visual appeal to showcase premium villas and industrial projects effectively.

## Core Design Elements

### Color Palette
**Light Mode:**
- Primary: 25 8% 15% (Deep charcoal for premium feel)
- Secondary: 220 15% 25% (Sophisticated navy)
- Background: 0 0% 98% (Pure, clean white)
- Text: 0 0% 10% (Rich black for readability)

**Dark Mode:**
- Primary: 45 25% 85% (Warm champagne accent)
- Secondary: 220 20% 75% (Soft blue-gray)
- Background: 220 15% 8% (Deep sophisticated dark)
- Text: 0 0% 95% (Crisp white)

### Typography
**Font Stack:**
- Primary: Playfair Display (Google Fonts) - Elegant serif for headlines and luxury branding
- Secondary: Inter (Google Fonts) - Clean sans-serif for body text and UI elements
- Accent: Montserrat (Google Fonts) - Modern sans-serif for property details and calls-to-action

**Hierarchy:**
- Hero titles: 3xl-4xl, Playfair Display, font-weight 700
- Section headers: 2xl-3xl, Montserrat, font-weight 600
- Body text: base-lg, Inter, font-weight 400
- Property details: sm-base, Inter, font-weight 500

### Layout System
**Spacing Framework:** Tailwind units of 4, 6, 8, 12, 16, 24 for consistent rhythm
- Container max-width: 7xl with generous horizontal padding
- Section spacing: py-16 to py-24 for premium breathing room
- Component spacing: Primary gaps of 6-8, secondary gaps of 4

### Component Library

**Navigation:**
- Fixed header with transparent background over hero, solid on scroll
- Primary navigation with hover effects and subtle animations
- Logo prominently displayed (left), contact CTA button (right)

**Hero Section:**
- Full-viewport height with premium property imagery
- Overlay with gradient (subtle dark overlay for text readability)
- Centered content with company tagline and primary CTA
- Minimal, elegant typography treatment

**Property Cards:**
- Clean, card-based layout with hover elevation
- High-quality imagery with subtle overlay on hover
- Property type badges (Villa/Industrial)
- Price and location prominently displayed
- "View Details" CTA buttons

**Gallery Components:**
- Masonry-style grid for property portfolios
- Lightbox functionality for detailed viewing
- Thumbnail navigation for property detail pages
- Smooth transitions and loading states

**Forms:**
- Sophisticated contact forms with floating labels
- Separate inquiry forms for villa vs. industrial projects
- Professional validation styling
- Premium button treatments

## Images
**Hero Image:** Large, stunning hero image featuring a luxury villa with Chilean landscape backdrop - full viewport height, professional architectural photography with dramatic lighting.

**Property Gallery:** High-resolution images of luxury villas showcasing interior and exterior shots, industrial project imagery highlighting scale and modern design.

**Background Elements:** Subtle texture overlays, professional photography of Chilean architecture and landscapes for section backgrounds.

**Visual Treatments:**
- Gradients: Subtle dark overlays (0 0% 0% 50% to transparent) on hero images
- Modern, sophisticated gradients using primary colors for accent elements
- Clean, architectural photography style throughout
- Consistent professional color grading on all property images

## Key Sections (Maximum 4)
1. **Hero:** Company introduction with luxury villa backdrop
2. **Portfolio:** Split showcase - Luxury Villas and Industrial Projects
3. **Company Heritage:** Brief about expertise in Chilean luxury construction
4. **Contact:** Premium inquiry forms with project-specific options

**Minimal & Impactful:** Front-loaded value with stunning hero, concise portfolio showcase, and streamlined contact process. Each section serves a specific conversion purpose without overwhelming visitors.
